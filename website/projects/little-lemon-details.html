Little Lemon

Project Details

For the Little Lemon Restaurant project, I developed a multi-page application as part of the finishing capstone project for the front-end developer course by Meta on Coursera. The front-end of this application was crafted using React.js, showcasing my ability to create responsive, dynamic interfaces. The design phase was particularly important in this project, as I was provided with only the logo and tasked with designing the entire user interface. Using Figma, I created a visually appealing and user-friendly layout that reflects the brand identity of the imaginary restaurant, highlighting my design skills and ability to translate concepts into functional digital experiences.

After completing the front-end, I expanded the project by developing the back-end, transforming it into a fully operational MERN stack application. Express was employed to build the backend, enabling seamless interaction between the front end and the database. This project not only demonstrates my proficiency in full-stack development but also highlights my ability to integrate design and functionality, creating a cohesive and fully functional application from concept to deployment. The Little Lemon Restaurant project stands as a testament to my skills in both front-end design and back-end development, showcasing my comprehensive capabilities as a developer.
