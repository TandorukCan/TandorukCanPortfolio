Fruit Slicing Game

Tech Stack

This project is a fruit slicing game developed using HTML for structure, CSS for styling, and jQuery for interactive features and dynamic content manipulation.

Project Details

The Fruit Slicing Game is an interactive browser-based game I developed using jQuery. In this game, the player begins with 3 lives and aims to achieve the highest possible score by slicing fruits that drop from the top of the screen. Each turn, a random fruit is dropped at a random speed, and the player must slice it before it falls off the screen. Successfully slicing a fruit destroys it and adds a point to the player's score, while missing a fruit results in the loss of a life.

The game continues until the player runs out of lives. At this point, the game ends, and the player is given the option to restart and try again. Additionally, the player can choose to reset the game at any time during gameplay, offering a fresh start. This project showcases my ability to create engaging and dynamic user experiences using jQuery, with a focus on responsive gameplay and intuitive user controls.
