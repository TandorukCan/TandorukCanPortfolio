Gazetteer App

Tech Stack

The front end was built using HTML, CSS, JavaScript, and Bootstrap, while PHP powered the back end. Additionally, several third-party APIs were integrated to fetch data.

Project details

This project involves developing a responsive, mobile-first website that provides detailed profiles for all countries, including demographic, climatic, geographical, and other relevant data. The website integrates multiple third-party APIs such as CountryStateCity, OpenWeatherMap, ExchangeRate-API, and Geonames to retrieve and display real-time information. The application features five buttons that allow users to view specific information about the chosen country, including general info, climatic data for the selected city or state, Wikipedia information, the latest news, and COVID-19 statistics.

Additionally, users can toggle marker clusters that represent parks, amusement parks, capital city/cities, universities, museums, and zoos within the selected country. The website utilizes the Leaflet library for dynamic mapping and overlays, enhancing user interaction through tools like Leaflet.markercluster and Leaflet.easyButton. Designed as a one-page, full-screen experience, the site functions seamlessly across all devices, leveraging frameworks like Bootstrap for streamlined design. A key feature is the ability to determine the user's current location and highlight their country on the map, with options to view additional data through modal dialogs. The project emphasizes a visually appealing and user-friendly interface, showcasing the developer's ability to deliver a superior, well-thought-out product. It also highlights the developer's skill in integrating diverse data sources to create a comprehensive and engaging user experience.
